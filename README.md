# CRUD SHOPPINGMALL Project - gwaja-sarang


## 🍪 프로젝트 소개
>> 중간 프로젝트 주제는 쇼핑몰로 정해졌고 쇼핑몰 틀 안에서 우리와 가장 친근했던 주제를 선택하려고 했습니다. 팀원들에게 투표를 실시한 후 가장 많은 표를 받았던 과자를 선택하게 되었습니다.
>구현 기능
-회원가입
-로그인(아이디 조회,비밀번호 찾기)
-전체 목록 출력
-검색(상세검색,키워드 검색)
-장바구니(수량조정,삭제,추가)
-주문(배송지 API) 
-모든 기능 DB 연결
